# /email — Hosted assets for email campaigns

These files are **not part of the website UI**. They are static assets hosted here
solely so they can be referenced by absolute URL in HTML email campaigns sent via ActiveCampaign.

| File | Used in |
|---|---|
| `gift-card-hero.webp` | Gift card announcement email — split-hero right panel |
| `gift-card.png` | Gift card announcement email — gift card mockup |

**Live URLs (via Vercel):**
- `https://soulfulsculpt.com.au/email/gift-card-hero.webp`
- `https://soulfulsculpt.com.au/email/gift-card.png`

Do not delete these files — they will break any sent or scheduled email campaigns that reference them.
